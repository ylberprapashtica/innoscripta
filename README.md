## About this app

Create the following alias

```bash
alias sail='sh $([ -f sail ] && echo sail || echo vendor/bin/sail)'
```

Then run, to start project

```bash
sail up -d
```

Migrate the database

```bash
sail artisan migrate
```

Populate data with minimal data

```bash
sail artisan db:seed
```

There's a Postman Collection in this project where you can test the API endpoints
